INTRODUCTION
------------

Query Builder is a module which build dynamic query for developers. 
Query Builder module helps the drupal developer especially those 
guys which are fresher in drupal.It reduces the amount of time for 
building query. It also provide options to clear cache, 
every time when cron run.


INSTALLATION
------------

The Query Builder module is very similar to other Drupal modules.
For installation of the Query Builder module 
please follow the below mentioned steps:

• Install as usual
• Enable the Query Builder module.


CONFIGURATION
-------------

• There is no configuration for this module.
